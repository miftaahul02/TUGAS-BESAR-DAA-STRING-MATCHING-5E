## ðŸ“– Deskripsi Singkat
Aplikasi ini mengimplementasikan algoritma **Knuth-Morris-Pratt (KMP)** untuk melakukan pencocokan string (*string matching*) yang efisien. Fitur utama aplikasi mencakup pencarian judul secara *real-time*, saran otomatis (*autocomplete*), serta penandaan teks (*highlighting*) pada hasil pencarian untuk memvisualisasikan cara kerja algoritma dalam menemukan kata kunci.
